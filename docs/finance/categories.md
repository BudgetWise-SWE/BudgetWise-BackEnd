# Financial Categories

The system utilizes categories to provide a taxonomical structure for all transaction data.

## Resource Endpoints

| Endpoint | Method | Functional Description |
|----------|--------|------------------------|
| `/api/finance/categories/` | `GET` | List all available categories |
| `/api/finance/categories/` | `POST` | Initialize a custom user category |

## System Defaults

The platform provides a standardized set of predefined categories. Users retain the capability to extend this taxonomy through the creation of custom, user-specific categories.
