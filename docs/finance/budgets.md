# Budgetary Controls

The system implements budgetary controls to enable users to monitor and constrain expenditures over defined periods.

## Budget Status Classifications

- **Active**: Current expenditure is within predefined limits.
- **Critical**: Expenditure has reached or exceeded 90% of the allocated limit.
- **Exceeded**: Cumulative expenditure has surpassed the total allocated limit.

## Granular Constraints

Users can implement specific spending constraints at the category level within a broader monthly budget. This allows for precise resource allocation across different sectors of expenditure.
